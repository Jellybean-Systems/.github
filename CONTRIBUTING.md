# Contributing to Jellybean Systems

Thanks for considering a contribution. We appreciate thoughtful engagement.

## Principles

1. **Transparency first** — If something involves AI, say so. Don't hide it.
2. **Privacy by design** — Don't collect data you don't need.
3. **Build in public** — Lessons learned, mistakes made, progress shared.

## How to Contribute

- **Issues** — Report bugs, suggest features, ask questions
- **Discussions** — Share ideas, get feedback before heavy work
- **Pull Requests** — Thoughtful changes with clear rationale

## Code Review

All PRs are reviewed by Saber and/or Jellybean. We'll be direct but fair.

## License

By contributing, you agree your work will be licensed under the same terms as the project (typically MIT).
